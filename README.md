# F1 EDA 2021 - Verstappen vs Hamilton

An exploratory data analysis of the 2021 Formula 1 World Championship —
one of the closest and most dramatic title fights in F1 history.

## What this project covers

| Chapter | Topic |
|---|---|
| 1 | Championship Points Progression |
| 2 | Qualifying vs Race Day Performance |
| 3 | Race Start Performance |
| 4 | Pit Stop Duration |
| 5 | Pit Stop Windows |
| 6 | Tyre Strategy Per Stint |
| 7 | Tyre Degradation & Pace Drop-off |
| 8 | Teammate Analysis |
| 9 | Safety Car & VSC Impact |
| 10 | Incidents, Crashes & Penalties |
| 11 | Fastest Lap Bonus Points |
| 12 | Sector-by-Sector Dominance |
| 13 | Telemetry & Racecraft |

## Tools & Libraries used

- Python
- FastF1
- Pandas
- Matplotlib
- Google Colab

## Dataset

Data is sourced via the [FastF1] (https://theoehrly.github.io/Fast-F1/) Python library,
which pulls official F1 timing and telemetry data.
